1. httpc.java : This is the httpc library file which is having all the httpc library operations.
2. MainDriver.java : This is the main driver file which will take all the commands from the console and provide option to exit the program.

Instructions to run the program:
1. You have to run the MainDriver.java file in order to run the program.
2. There are 2 operations that you can do. 
  a) Enter Curl Command. : You can enter anny of the valid curl commands. The command should start with httpc rather than Curl.
  b) Exit the process.  This will exit you from the application.
